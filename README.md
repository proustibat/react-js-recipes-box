http://prstbt-recipes-box.surge.sh/

## Installation and usage

```
npm i
npm start
```

## Build
```
npm run build
```

## Deployment
It uses Surge (http://surge.sh/), you can change npm script to deploy on other platforms (github pages for example)
```
npm run deploy
```
